# Абстрактная интерпретация на Rocq

Статические анализаторы пытаются автоматически установить некоторые свойства
программ без их выполнения.

Статические анализаторы можно использовать по-разному, например:
- для поиска ошибок, допускаемых программистами;
- в качестве верификаторов, гарантирующих соответствие программы определенным
  свойствам.

При поиске ошибок анализ должен быть точным (слишком большое количество ложных
срабатываний делает инструмент непригодным), однако не ожидается и не
предоставляется гарантия обнаружения всех ошибок определенного класса. С другой
стороны, при верификации программ *корректность анализа* имеет первостепенное
значение: если анализатор не выдает предупреждений, программа должна быть
свободна от класса отслеживаемых анализатором ошибок.

Наша задача - разработать корректный статический анализатор для простого
императивного языка программирования. Для этого будем использовать абстрактную
интерпретацию, а доказательство корректности будем проводить с помощью Rocq.
